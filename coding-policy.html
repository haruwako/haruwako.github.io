HTML

  • ・HTML5にて記述します。
  • ・印刷は環境に依存するため、原則考慮しません。
  • ・Bootstrap等のフレームワークを使用する場合があります。
  • ・文法に注意し、第三者に見やすいコードを心がけます。

CSS

  • ・SCSSを用いて記述します。
  • ・印刷は環境に依存するため、原則考慮しません。
  • ・基本的にrem単位を用います。
  • ・文法に注意し、第三者に見やすいコードを心がけます。

Java Script

  • ・jQueryとプラグインを主に使用します。
  • ・jQueryが無効となるブラウザは考慮致しません。
  • ・文法に注意し、第三者に見やすいコードを心がけます。

Quality Control

  • ・クロスブラウザ対応、W3C準拠をチェック後納品致します。
  • ・IE以外は製作時の最新バージョンに準拠します。
  • ・IE 10~11対応で、IE9以下のバージョン対応は考慮しません。
  • ・iPhone実機にてレスポンシブ対応の確認をします。
  • ・Androidはエミュレータでレスポンシブ対応の確認をします。
